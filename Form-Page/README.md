# Registration Form

This project is a simple **HTML & CSS registration form** built as part of my HTML learning exercises.

It collects basic user information and the services a user is interested in, using form elements such as text inputs, email inputs, checkboxes, and a submit button.

---

##  Project Features

- Registration form layout inside a styled card
- First name and last name input fields
- Email address input with validation
- Company / organization input field
- Website input field
- Multiple service options using checkboxes:
  - Social media banners/posts
  - Website design/redesign
  - E-commerce/shop design
  - Photography/video
  - Email marketing/design
- Submit button to send the form

---

##  Technologies Used

- **HTML5** – page structure and form elements  
- **CSS** – styling and layout (external stylesheet)

---

## Notes

1. This project is frontend-only (no backend or database).

2. It is intended for learning and practice purposes.

3. Form submission does not send data anywhere.
